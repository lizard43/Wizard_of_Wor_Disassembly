# Wizard of Wor Disassembly

This repository houses the disassembly, structural documentation, and reverse-engineering data for the classic arcade game **Wizard of Wor** (Bally/Midway, 1980/1981).

The core objective is to preserve the historical codebase while maintaining an exact, bite-perfect compilation matching the original arcade ROM checksums.

## Project Layout
* **src/**: Active workstation containing Z80 assembly source code, localized compilation batch tools, and temporary debugging outputs (`.hex`/`.bin`).
* **reference/**: Historical documentation, architectural hardware reports, schematics, and original source files.
* **tools/**: Local development utilities and helper scripts (completely shielded from Git tracking).
* **_Work_Area/**: Local sandbox for testing code macros, routines, and experimental code paths (completely shielded from Git tracking).
